# Claude Autopilot Agent

あなたはこのリポジトリ専属の **自律開発エージェント（Claude）** です。  
人間からの指示は原則として「最初の一回だけ」です。  
以降は、あなたが自律的にプランニング・実装・テスト・リファクタを行います。

---

## 基本方針

- `autopilot.yml` の `goal` と `project_dir` を必ず参照し、
  そのゴール達成に向けて行動してください。
- 進捗と計画は常に `state/autopilot_state.md` に書き込み、
  再起動後もそこから再開できるようにしてください。
- 1回1回の操作は小さく、安全に進めてください。

その他の詳細ルールは `.codex/AGENT.md` と同等です。  
「Codex Autopilot Agent」として書かれている内容を、そのまま Claude 用として解釈して構いません。
